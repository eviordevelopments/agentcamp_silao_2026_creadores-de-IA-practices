# Agent Camp — Guía y Recursos

**Descripción**

Agent Camp es un conjunto de prácticas, guías y prompts diseñados para enseñar y acelerar la creación de agentes de IA prácticos. Este repositorio contiene materiales de referencia, talleres y colecciones de prompts en español para facilitar el aprendizaje y la implementación.

**Objetivos**

- Proveer una guía paso a paso para crear y experimentar con agentes de IA.
- Reunir prompts especializados y plantillas reutilizables.
- Ofrecer ejercicios y talleres para aplicar conocimientos en proyectos reales.

**Cómo usar este repositorio**

1. Explore los documentos principales (índice y guías) para entender la estructura.
2. Abra los notebooks y archivos de taller para seguir los ejercicios prácticos.
3. Reutilice y adapte los prompts en `PROMPTS_*.md` según sus necesidades.

**Estructura del repositorio (resumen)**

- `INDICE_MAESTRO_AGENTCAMP.md`: Índice general del contenido.
- `guia_implementacion_paso_a_paso.md`: Guía detallada de implementación.
- `notebooks_codigo_agentcamp.md`: Notebooks de ejemplo y código.
- `PROMPTS_ANTIGRAVITY_ESPECIALIZADOS.md`: Prompts especializados.
- `PROMPTS_MAESTROS_COPIAR_PEGAR.md`: Prompts listos para usar.
- `TALLER_AGENTCAMP_90MIN_RETO30DIAS.md`: Taller y plan de reto.
- `README.md`: (Versión base/ingles u otra) — consulte para detalles adicionales.

**Requisitos mínimos**

- Sistema operativo: Linux / macOS / Windows
- Python 3.10+ (si usa notebooks o scripts Python)
- Entorno virtual recomendado: `venv` o `conda`

**Instalación rápida (ejemplo Python)**

```bash
python -m venv .venv
source .venv/bin/activate
pip install -r requirements.txt  # si existe
```

**Uso recomendado**

- Lea primero el `INDICE_MAESTRO_AGENTCAMP.md` para ubicar recursos.
- Siga `guia_implementacion_paso_a_paso.md` para proyectos de inicio.
- Ejecute los notebooks listados en `notebooks_codigo_agentcamp.md` para ejemplos prácticos.
